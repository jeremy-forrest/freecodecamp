# Travel Agency Page

A simple travel agency webpage showcasing Asheville, North Carolina as a travel destination.

## Overview

This project is an HTML lab exercise that demonstrates basic web development skills including semantic HTML structure, images with figure elements, lists, and links.

## Features

- **Destination Showcase**: Highlights Asheville, North Carolina with descriptive content
- **Travel Packages**: Links to group travel and private tour options
- **Top Itineraries**: Three featured destinations with images and captions:
  - Blue Ridge Parkway
  - Biltmore Estate
  - Downtown Asheville
- **Semantic HTML**: Uses proper HTML5 elements like `<figure>`, `<figcaption>`, and semantic headings
- **Accessibility**: Includes meta descriptions, alt text for images, and proper link targets

## Technologies Used

- HTML5
- Semantic markup
