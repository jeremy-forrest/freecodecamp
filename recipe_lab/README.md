# Recipe Lab

A simple HTML recipe page for creamy scrambled eggs, created as part of the freeCodeCamp curriculum.

## Description

This project demonstrates basic HTML structure including headings, paragraphs, lists, and images. The page contains a complete recipe with ingredients and step-by-step cooking instructions.

## Recipe

**Creamy Scrambled Eggs** - A guide to making perfectly creamy scrambled eggs with optional garnishes.

## File Structure

- `recipe lab.html` - Main HTML file containing the recipe page

## How to View

Open `recipe lab.html` in any web browser to view the recipe page.

## Technologies Used

- HTML5

## Learning Objectives

This project demonstrates:
- Semantic HTML structure
- Unordered lists (`<ul>`) for ingredients
- Ordered lists (`<ol>`) for step-by-step instructions
- Images with alt text for accessibility
- Basic metadata and viewport configuration
